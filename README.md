# rsn8d_data-

A lightweight browser-based data tool for quick streaming and PDF export.
Built for simplicity, sharing, and easy use in student and physics workflows.

Created by **CreizyLabs**.

## Why this exists

I made this because I wanted a minimal tool that just works — no heavy setup, just open the file, stream data, and export.

## Installation / Usage

1. Download or clone this repository.
2. Open `rsn8d.data.html` in any modern web browser.
3. Stream your data.
4. Export to PDF when needed.

That’s it.
